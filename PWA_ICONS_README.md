# PWA Icon Instructions

To complete the PWA setup, you need to add the following icons to the `public` folder:

## Required Icons:

1. **icon-192x192.png** - 192x192px icon (can be the Film/Trakt logo)
2. **icon-512x512.png** - 512x512px icon (same design, higher resolution)

## Optional Screenshots for App Store:

3. **screenshot-wide.png** - 1280x720px (desktop view screenshot)
4. **screenshot-narrow.png** - 750x1334px (mobile view screenshot)

## How to create these icons:

### Option 1: Use an online tool
- Visit https://www.pwabuilder.com/imageGenerator
- Upload a square logo (at least 512x512px)
- Download the generated icons

### Option 2: Manual creation
- Create a square image with the Film icon (ðŸŽ¬) or app logo
- Background: #7c3aed (purple) or gradient
- Export as PNG in the required sizes

### Recommended design:
- Purple gradient background (matching theme color)
- Film icon in white or light color
- Centered and with padding
- Clear and recognizable at small sizes

Place all generated icons in the `public` folder.
